### Detailed Explanation 1. **Defining Variables**: Variables are defined within a scope. The most common place to define them globally is within the `:root` pseudo-class. ```css :root { --main-bg-color: lightblue; --main-text-color: darkblue; --main-padding: 10px; } ``` - `:root` is a pseudo-class that matches the document's root element (typically ``). - Variables are prefixed with `--`, followed by the variable name and a value. 2. **Using Variables**: Variables are accessed using the `var()` function. ```css body { background-color: var(--main-bg-color); color: var(--main-text-color); padding: var(--main-padding); } ``` - `var(--variable-name)` is used to access the value of the variable. - Variables can be used anywhere in the CSS where values are accepted. 3. **Reusing Variables**: You can reuse variables multiple times throughout your CSS. ```css .container { background-color: white; padding: var(--main-padding); } .highlight { color: var(--main-bg-color); } ``` - This reduces redundancy and makes the CSS easier to maintain. ### Advanced Features #### Fallback Values CSS variables can have fallback values in case the variable is not defined. ```css body { background-color: var(--undefined-variable, pink); /* Use 'pink' if --undefined-variable is not defined */ } ``` #### Changing Variables Dynamically CSS variables can be changed dynamically using JavaScript, making them powerful for themes or responsive designs. ```html Dynamic CSS Variables ``` ### Detailed Explanation 1. **Transition**: Adding a transition effect to smooth the change of background and text color. ```css body { transition: background-color 0.5s, color 0.5s; } ``` 2. **JavaScript Interaction**: Using JavaScript to change the variable values. ```javascript document.getElementById('change-theme').addEventListener('click', () => { document.documentElement.style.setProperty('--main-bg-color', 'darkblue'); document.documentElement.style.setProperty('--main-text-color', 'lightblue'); }); ``` - `document.documentElement` refers to the root element (``). - `style.setProperty('--variable-name', value)` changes the value of the CSS variable. ### Conclusion CSS variables are a powerful tool for managing CSS values, reducing redundancy, and making dynamic changes easier. By defining variables in a central place and reusing them, you can keep your CSS cleaner and more maintainable. Using JavaScript to change these variables allows for dynamic theming and responsiveness in your designs.
